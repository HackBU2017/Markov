Who are we?

We are a group of Binghamton University students. This project started as a piece of the 2017 HackBU Hackathon but will be deployed in the future as a full service site.

How does it work?

Our model uses python and a Twitter API to collect data from various twitter handles as they are requested. We then use a python code to run a Markov chain on the text. The Markov chain uses the relative probablity of different word combinations to create full sentances or "tweets" based off of past tweeting behaviour.
